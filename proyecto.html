Proyecto 1: Sistema de Gestión de Tareas

Desarrollé un sistema para la gestión de tareas que permite a los usuarios organizar sus actividades de manera eficiente.

Proyecto 2: Página Web Personal

Creé una página web personal para mostrar mis habilidades y proyectos, utilizando HTML5 y buenas prácticas de accesibilidad.

Proyecto 3: Aplicación educativa e interactiva para niños de 1 a 5 años

Creé una aplicación interactiva para niños, que incluye un temporizador que alerta a los padres cuando el tiempo de uso ha caducado.

Proyecto 4: Calculadora Básica (BAISICAL)

Esta calculadora fue diseñada para la materia de lógica, permitiendo conversiones binarias, octales y hexadecimales.

Proyecto 5: Prototipos de carros eléctricos con materiales reciclados

Estos prototipos fueron creados con materiales reciclados, baterías y motores, para demostrar la creatividad y apoyar el medio ambiente.
